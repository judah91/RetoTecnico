# Microservicios de Transacciones con Kafka y PostgreSQL
Este proyecto implementa dos microservicios (`TransactionService` y `AntiFraudService`) que se comunican mediante Kafka para procesar transacciones financieras.

## Tecnologías Usadas
- .NET 8
- PostgreSQL
- Kafka
- Docker & Docker-Compose

## Cómo Ejecutar
Ejecuta los siguientes comandos en la terminal:
docker-compose up --build
